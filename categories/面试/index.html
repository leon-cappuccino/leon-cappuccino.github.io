Android面试题之四大组件

2022年3月10日
| 面试 Android基础
Android四大组件相关 一、Activity相关 1.介绍不同场景下的Activity生命周期变化过程？ (1)Activity正常启动 onCreate->onStart->onResume (2)Activity启动另一个Activity 1)B完全挡住A A:onPause -> B:onCreate -> B:onStart -> B:onResume -> A:onSaveInstanceState-> A:onStop 如果我们在此时点击Back键，将依次执行B:onPause -> A:onRestart -> A:onStart -> A:onResume -> B:onStop -> B:onDestroy 2）B没有完全挡住A，如dialog A:onPause -> B:onCreate -> B:onStart -> B:onResume -> A:onSaveInstanceState 可以看到，相比完全挡住的情况，由于A可见，因此不会到onStop 如果我们此时点击Back键，将依次执行B:onPause -> A:``onResume -> B:onStop -> B:onDestroy` 如果我们此时点击home键，将依次执行B:onPause -> A:onStop-> B:onSaveInstanceState -> B:onStop，再将应用切到前台 A:onRestart -> A:onStart -> B:onRestart -> B:onStart -> B:onResume 注意以上的讨论前提是启动模式为standard。 (3)Activity按back键返回 onPause -> onStop -> onDestroy (4)按home键返回到桌面 onPause -> onSaveInstanceState -> onStop……
